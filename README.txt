#  Template para Fotógrafos

Obrigado por adquirir este template!
Este site foi desenvolvido para fotógrafos que desejam ter uma presença profissional online de forma rápida e simples.

---

#  Como usar

1. Extraia os arquivos do projeto
2. Abra o arquivo **index.html** no navegador
3. Edite os arquivos conforme necessário

---

#  Estrutura de pastas

* **index.html** → Estrutura principal do site
* **/css/style.css** → Estilos do site
* **/js/script.js** → Scripts (animações e interações)
* **/img/** → Imagens do site

---

#  Como editar o site

##  Alterar textos

Abra o arquivo **index.html** e edite os textos diretamente.

Exemplo:

```
<h1>Seu Nome</h1>
```

---

##  Trocar imagens

1. Vá até a pasta **/img/**
2. Substitua as imagens existentes
3. Mantenha o mesmo nome OU altere no HTML

Exemplo:

```
<img src="img/sua-imagem.jpg">
```

---

##  Alterar WhatsApp

No arquivo **index.html**, procure por:

```
https://wa.me/5599999999999
```

Substitua pelo seu número com DDD.

Exemplo:

```
https://wa.me/559900789978
```

---

#  Personalização

Você pode alterar:

* Cores no arquivo **style.css**
* Fontes
* Imagens
* Textos

---

#  Dicas importantes

* Use imagens de boa qualidade
* Prefira fotos no formato **.webp** para melhor performance
* Teste no celular antes de publicar

---

#  Como publicar o site

Você pode hospedar gratuitamente em:

* Netlify
* Vercel

Basta fazer upload dos arquivos.
